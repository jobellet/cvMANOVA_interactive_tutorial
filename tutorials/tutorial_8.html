Power Analysis

Optimizing sample size and searchlight radius for cvMANOVA.

Number of Voxels ($p$) 123

Typical searchlight (r=3.0) ≈ 123 voxels.

Number of Trials ($n$) 40
True Effect Size ($D$) 1.5
$\text{Var}(\hat{D}) \approx \frac{p \cdot \sigma^2}{n}$

Rank-Deficiency Warning

The number of voxels ($p$) exceeds 90% of your error degrees of freedom ($df_{error}$). The covariance matrix is now rank-deficient. To proceed, **Regularization (Shrinkage)** would be required, which introduces bias and prevents $\hat{D}$ from being a strictly unbiased estimator.

85% Statistical Power
0.24 Standard Error (Noise)

Understanding the Trade-offs

1. Variance scales with $p$

Observe how the distribution of $\hat{D}$ widens as you increase the Number of Voxels. Under the null hypothesis, the sampling variance of pattern distinctness is approximately proportional to $p$. Larger searchlights include more information but also more null noise.

2. Data volume reduces error

Increasing the number of trials or independent runs decreases the variance of the estimate. This tightens the distribution curve around the true value, improving the signal-to-noise ratio and boosting statistical power.
