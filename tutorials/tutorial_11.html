The Code Bridge

Interoperable cvMANOVA implementation side-by-side.

Consistency Note

Ensure your first-level model uses AR(1) or FAST temporal autocorrelation modeling. Without this, the noise covariance estimate ($\Sigma$) will be biased, leading to inaccurate pattern distinctness ($D$) values.

M MATLAB (SPM) 
% 1. Set Path and Parameters
resultsDir = './stats/results';
slRadius   = 3; % Radius in voxels (~123 vox)

% 2. Specify Contrast
% SPM model MUST model temporal correlations (AR1/FAST)
% to correctly estimate pattern distinctness
Cs = [1 -1]; 

% 3. Searchlight Analysis
permute = 0;
cvManovaSearchlight(resultsDir, slRadius, Cs, permute);

% 4. ROI Analysis
maskFile = 'visual_cortex.nii';
cvManovaRegion(maskFile, resultsDir, Cs);

% Output: cms_Ds_c*p*.nii (D-statistic)

P Python (Nilearn) 
from cvmanova_python import SearchlightCvManova
from cvmanova_python.loaders import SPMLoader
from cvmanova_python.design import ContrastSpec

# 1. Load Data
loader = SPMLoader('./stats_dir')
X, Y, runs = loader.load_data()

# 2. Factorial Contrast Design
cs = ContrastSpec(factors=['condition'], levels=[2])
contrast = cs.get_interaction()

# 3. Initialize Searchlight Estimator
sl = SearchlightCvManova(radius=3, contrast=contrast)

# 4. Compute Pattern Distinctness
results = sl.fit(X, Y, groups=runs)

# 5. Extract results
peaks = results.get_peaks(threshold=3.1)
results.to_nifti('./manova_map.nii')

from nilearn.plotting import plot_glass_brain
plot_glass_brain('./manova_map.nii')
