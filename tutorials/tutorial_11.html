Bridge: MATLAB & Python

Implementation patterns across the cvMANOVA ecosystem.

Technical Requirement

Whether using MATLAB or Python, the underlying first-level GLM must model temporal autocorrelations (typically using AR(1) or FAST models) to ensure valid pattern distinctness estimates.

MATLAB (SPM) 
% Load SPM model (Ensure AR(1)/FAST is used!)
load('SPM.mat');

% Define Searchlight Parameters
dirName  = './results_cvmanova';
slRadius = 3; % ~123 voxels
permute  = 0;

% Define Contrast Matrices
C1 = [1 -1 0 0]; 
Cs = {C1};

% Execute Searchlight
stats = cvManovaSearchlight(dirName, slRadius, Cs, permute);

% Results generated: cms_Ds_c1p1.nii
% Standardized Pattern Distinctness Map

Python (Nilearn) 
from cvmanova import SearchlightCvManova, SPMLoader
from cvmanova.contrasts import ContrastSpec

# Load from SPM directory
loader = SPMLoader('./first_level_results')
X, Y, mask = loader.load()

# Define Contrast
c_spec = ContrastSpec(conditions=['A', 'B'])
contrast = c_spec.difference('A', 'B')

# Run Searchlight
sl = SearchlightCvManova(radius=3.0)
sl.fit(X, Y, mask=mask, contrast=contrast)

# Export Results
sl.to_nifti('searchlight_D.nii')
sl.plot_glass_brain()
