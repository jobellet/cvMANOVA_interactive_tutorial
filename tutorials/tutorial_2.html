The Pattern Reliability Metric D

Drag conditions in voxel space to intuitively build the cvMANOVA distinctness statistic.

Pattern distinctness: D = trace( B' X'X B Σ⁻¹ ) / n

D compares the between-class variance (how far apart conditions are) to the within-class variance (the noise structure). The larger D, the more reliably distinct the neural patterns are.

Interpretation:
D ≫ 0 → conditions are reliably distinct
D ≈ 0 → under null hypothesis
D < 0 → noise artifact (expect ~50% of the time when true D = 0)

What drives D?

Between-class Distance ↑

Moving the centroids farther apart increases D. The brain encodes distinct states reliably.

Within-class Variance ↑

More noise reduces D. Even a large true effect becomes undetectable in high-noise regimes.
